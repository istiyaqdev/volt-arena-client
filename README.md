## 🌟 Overview

VoltArena is a full-stack gaming venue booking platform that connects gamers, esports teams, and gaming communities with premium gaming spaces.

Whether you're looking to reserve a high-end gaming station, book an esports arena for scrims, host a LAN event, or organize a tournament, VoltArena makes the process seamless, fast, and reliable.

## 📦 Environment Variables

### Frontend

```env
NEXT_PUBLIC_API_URL=http://localhost:5000
BETTER_AUTH_SECRET=your_secret_here
BETTER_AUTH_URL=http://localhost:3000
MONGODB_URI=your_mongodb_connection_string
```

### Backend

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
CLIENT_URL=http://localhost:3000
```
